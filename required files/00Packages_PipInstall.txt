Installation:
First of all, install BeautifulSoup and Requests. You can simply use pip in Terminal (on Linux/Mac) or CMD (on Windows) as follows:

pip install beautifulsoup4

pip install requests

pip install fake-useragent

If you are on Linux or Mac, starting with sudo might be required.

Installing fake_useragent Package

To install fake_useragent Python package, you can simply use pip:

pip install fake-useragent 

You can use it as

from fake_useragent import UserAgent
ua = UserAgent()

Note: When you install using pip, the separator is a dash fake-useragent  but when you import the package, the separator is underscore fake_useragent
